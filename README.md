* **Context:** 1학년 학기 과제로 진행한 팀 프로젝트 (4인 1조)
* **Service Overview & Vision**

    * **Future Scope (AI Healthcare):** 누적된 보행 패턴 데이터를 머신러닝으로 분석하여, 척추 불균형 등 신체 자세 문제를 진단하는 헬스케어 솔루션으로의 확장성을 고려하여 설계
* **My Contribution (Architecture & Direction)**
    * **Technical Decision:** 추후 AI 도입을 고려해 방대한 시계열(Time-series) 데이터 적재에 유리한 InfluxDB 도입을 제안하고, 백엔드 팀에게 스키마 설계 가이드라인 제시
    * **Protocol Design:** Hardware(ESP32)와 Server 간의 데이터 전송 주기 및 패킷 구조(Protocol)를 직접 설계하여 통신 효율 최적화
    * **Project Management:** 개발 단계별 마일스톤 설정, 파트별(HW/Back/Front) 구현 수준 및 테스트 데드라인 관리
* **Development (Frontend)**
    * **Web-based App:** TypeScript와 React(Vite) 를 기반으로 사용자에게 직관적인 데이터 대시보드 웹 애플리케이션 개발
* **Stack:** `C++` `ESP32` `TypeScript` `React` `InfluxDB`
